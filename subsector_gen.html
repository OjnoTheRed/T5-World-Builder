Subsector Generator

This is a quick-and-dirty demonstration of setting up a subsector generator that links each randomly generated world to TravellerWorlds for further details. The names are a fixed set that I randomly generated using an online random word generator that uses Markov chains.

Future versions will include the ability to modify the density, edit all the values, save/load and export/import sectors you have worked on, and eventually generate a map using TravellerMap.
