Trade Codes and Remarks

Calculated by the engine, not entered by the user.

{Ix}

T5 World Importance from -4 to +5. Ignored in map and system creation.

(Ex)

T5 Economic Extension; e.g. 'A73-1'. Used for placing Resource hexes on maps.

[Cx]

e.g. T5 Cultural Extension; e.g. '9717'. Ignored in map and system creation.

Nobility

T5 Nobles present; e.g. 'ecB'. Ignored in map and system creation; a Noble Estate is always placed.

Bases

Bases in T5 format; e.g. 'S' or 'NS'. Ignored in map creation and system creation.

Zone

F/U=Forbidden/Unabsorbed (non-Imperial)

Multiplier

Number of planets other than the main world and gas giants. Ignored in map creation. This number of worlds are placed in system generation.

Interstellar government code. Ignored in map and system creation.

3. Random Seed

Random number seed

Seed is an integer between 0 and 4294967295. Leave blank for a random seed.

4a. Map Options

Black and White Map

Place a Noble Estate on Map

If this is checked, the World Mapper will place a Noble Estate if and only if the Noble field is not blank. If it is unchecked, a noble estate will not be placed.

Place estates for each noble

If this is checked, the World Mapper will place one Noble Estate for each Noble in the Nobles field. When the user generates a World Hex, the correct number of Terrain Hexes for that Noble will be placed (e.g. Baron will have 4).

Allow estates to be placed in oceans

If this is checked, the World Mapper will allow Noble Estates to be placed in Ocean hexes on the World Map and on hex maps. If unchecked, noble estates will be places on land or islands.

Number of world maps to generate:

Generate up to 10 maps at once. The first map will use the seed specified in the random seed box on this page. Remaining seeds are picked randomly. Each map has its URL provided.

Size:

If you just want a blank world map to download and play with yourself. Pick a size and click the button.

4b. System Options

Main World is a satellite

Force the placement of the Main World as a sattelite of a gas giant or a big world if no giants.

Main World is not a satellite

Force the placement of the Main World as a planet not a satellite. If both satellite options are checked, the Main World will be placed as a satellite. If both are unchecked, the T5 rules will be used to determine placement.

Force placement of MW in HZ

If this is unchecked, the T5 rules will be used to place the Main World in HZ-1, HZ or HZ+1. If checked, the Main World will be placed in the HZ.

Worlds tidally locked to star have no satellites

If this is unchecked, the T5 rules will be used determine the number of satellites. If checked, and a world is tidally locked to its star, it will have no satellites.

If mainworld population is zero, all other worlds are barren.
